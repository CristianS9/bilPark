###################
Proyecto final de FP superio: Parking en bilbao con mapa estatico con imagenes.
###################

Repositorio que contiene el proyecto final de fp superio Desarrollo de applicaciones web. Es de pega, el bueno es el repositorio eTickets previo a este.

El back-end de este repositorio este hecho con PHP y la como core con la libreria CodeIgniter.
Utiliza una BBDD de MariaDB (MySql) cuyo fichero esta en el repositorio.


*******************************
Configuracion para que funcione
*******************************

	- Establecer este repositorio en el directorio de ejecucion de un Apache con PHP integrado
	- Importar el fichero parking.sql que se encuentra el el directorio root del repositorio en una BBDD MariaDB(MySql)
	- Configurar el fichero application/config/config.php con lo datos de la BBDD utilizada
